# 2048游戏

这是一个使用Python和Kivy框架实现的2048游戏。游戏提供了简洁的图形界面和流畅的操作体验。

## 功能特点

- 经典的2048游戏玩法
- 支持键盘方向键控制
- 实时分数显示
- 支持新游戏重置
- 游戏结束提示
- 完整的中文界面

## 安装说明

1. 确保你的系统已安装Python 3.x
2. 安装依赖包：
   ```bash
   pip install -r requirements.txt
   ```

## 运行游戏

```bash
python main.py
```

## 游戏操作

- 使用键盘方向键（↑↓←→）控制方块移动
- 点击"新游戏"按钮开始新的游戏
- 当无法继续移动时游戏结束

## 开发环境

- Python 3.12.3
- Kivy 2.3.1

## 系统要求

- Windows/Linux/MacOS
- Python 3.x
- 显示分辨率：建议1280x720或更高
